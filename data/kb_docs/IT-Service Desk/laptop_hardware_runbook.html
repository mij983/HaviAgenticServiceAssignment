Laptop and Hardware Issues - IT Service Desk Runbook

This runbook covers common laptop and hardware issues handled by the IT Service Desk. Use it to triage and route tickets correctly.

Laptop Not Turning On

Symptoms: Screen is blank, no power LED, or device powers off immediately.

  • Hold the power button for 10 seconds to force power off, then restart
  • Check if the charger is connected and the power outlet is working
  • Remove any connected USB devices or docking stations and retry
  • If still not turning on — raise a hardware replacement request with IT-Service Desk

Blue Screen of Death (BSOD)

Symptoms: Windows shows a blue screen with an error code and restarts.

  • Note the STOP error code from the blue screen (e.g. KERNEL_SECURITY_CHECK_FAILURE)
  • Check if the issue started after a Windows update — if yes, escalate to IT-Wintel Support
  • Run Windows Memory Diagnostic: Start > search "Windows Memory Diagnostic"
  • If BSOD is frequent (more than twice a week), escalate to IT-Wintel Support for driver/OS investigation

Laptop Running Slowly

Symptoms: Applications take a long time to open, system is unresponsive.

  • Check disk usage in Task Manager — if disk is at 100%, restart the machine
  • Check if a Windows update or antivirus scan is running in the background
  • If the issue persists after restart, escalate to IT-Wintel Support

Cannot Connect to Wi-Fi

Symptoms: Device shows no Wi-Fi networks, or fails to connect to corporate SSID.

  • Toggle the Wi-Fi switch off and on
  • Forget the corporate network and reconnect
  • If the Wi-Fi adapter is missing in Device Manager — escalate to IT-Wintel Support
  • If corporate Wi-Fi is down in the office — escalate to IT-Network Support

Printer Not Working

Symptoms: Print jobs are stuck in queue, printer shows offline.

  • Restart the Print Spooler service: Run > services.msc > Print Spooler > Restart
  • Delete stuck jobs from the print queue
  • Re-add the printer using the corporate print server: \\printserver\printername
  • If the printer itself is physically faulty — raise a hardware ticket with IT-Service Desk
